# 飞机大战 Plane Battle

经典纵向卷轴射击游戏( shmup )的网页版。
纯 HTML5 Canvas + 原生 JavaScript 实现,零依赖、零构建,打开即玩。

## 在线试玩

https://games.leonova.xyz/plane-battle/

## 本地运行

无需安装任何东西,直接用浏览器打开 `index.html` 即可。
也可以起一个本地服务器:

```bash
npx serve .
```

## 操作方式

### 电脑(键盘)

| 按键 | 功能 |
| --- | --- |
| 方向键 / WASD | 移动战机(支持 8 向) |
| ← / →(标题界面) | 切换战机 |
| B | 放雷(冲击波清空敌弹 + 全屏伤害) |
| P | 暂停 / 继续 |
| Enter | 开始 / 重新开始 |

射击为**自动开火**,只需专注走位。

### 平板 / 手机(触屏)

- **按住画面拖动**即可移动战机,飞机会定位在触点上方,避免被手指遮挡。
- 标题界面:点画面**左右两侧**切换战机,点中间开始。
- 右下角 **💣** 按钮放雷。

## 玩法说明

- 驾驶战机不断向上突进,击毁敌机得分,被敌弹或敌机撞到则损失一条命。
- 共 3 条命,全部阵亡游戏结束;阵亡后会在出生点带短暂无敌保护罩重生。
- 生存时间越长,敌机出现越快、越密集,中/大型占比越高。
- 每 90 秒登场一个 **Boss**:顶部巡游、扇形弹幕,击毁得 3000 分并掉落 2 个道具。
- 战机选择与最高分都会保存在浏览器本地。

### 可选战机

三种战机手感与弹种各不相同,在标题界面切换:

| 战机 | 定位 | 武器形态(Lv1 → Lv4) |
| --- | --- | --- |
| **猎鹰** | 均衡型,直射光弹 | 单发 → 双发 → 三向 → **五向齐射** |
| **阿帕奇**(武装直升机) | 火力型,移动稍慢 | 机炮 → 双机炮+1 导弹 → 双机炮+2 导弹 → **三向机炮+双联追踪导弹** |
| **幻影**(飞翼) | 速度型,移动更快 | 穿透波弹 → 双发 → 三向 → **强化波弹**(更大、双倍伤害,穿透所有敌机) |

### 敌方单位

| 类型 | 特点 | 分数 |
| --- | --- | --- |
| 红色小型机 | 1 HP,速度快,不射弹 | 100 |
| 紫色中型机 | 4 HP,直线弹 | 300 |
| 墨绿大型机 | 10 HP,双发直线弹,必掉道具 | 800 |
| Boss 炮艇 | 高 HP,扇形三连发 | 3000 |

### 道具

| 道具 | 效果 |
| --- | --- |
| ⭐ 武器升级 | 提升一级武器形态(满级 Lv4),满级后折算 500 分 |
| 💣 雷 | 雷数 +1(上限 5),满后折算 500 分 |
| ❤️ 命 | 命数 +1(上限 5),满后折算 1000 分 |

### 放雷

放雷会以战机为中心炸开**金/青双色冲击波**,伴随金色粒子、白闪与震屏,
瞬间清空所有敌弹并对全屏敌机(含 Boss)造成高额伤害。

## 技术要点

- 纯 Canvas 矢量手绘图形与 WebAudio 合成音效,无任何外部素材与依赖。
- 三种战机共用一套状态机,仅弹种/移速/射速不同;追踪导弹按有限转角
  转向最近目标,穿透波弹用命中集合保证同一目标只结算一次伤害。
- 三层视差星空滚动背景;爆炸粒子、屏幕震动、放雷冲击波等反馈效果。
- 玩家受击判定为远小于贴图的小圆(弹幕游戏惯例),手感更宽松。
- 难度曲线随存活时间推进:刷怪间隔、敌机速度、中/大型占比递增。
- `?demo=秒数&plane=机型` 可直接开局并快进,用于预览截图与演示。

## 文件结构

```
index.html        入口页面
style.css         页面样式
js/audio.js       WebAudio 音效
js/waves.js       常量与难度曲线/出怪表
js/sprites.js     矢量绘制函数(三种战机/弹种/敌机/道具/Boss)
js/entities.js    玩家机 / 敌机 / Boss / 子弹 / 道具 / 粒子
js/game.js        游戏状态机、碰撞、计分、HUD、主逻辑
js/main.js        键盘 + 触屏拖动输入与主循环
test/simulate.js  Node 无头冒烟测试: node test/simulate.js
```
