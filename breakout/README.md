# 打砖块 Breakout

经典弹球打砖块的网页版,含多球、激光、变长板道具与循环关卡。
纯 HTML5 Canvas + 原生 JavaScript 实现,零依赖、零构建,打开即玩。

## 在线试玩

https://games.leonova.xyz/breakout/

## 本地运行

无需安装任何东西,直接用浏览器打开 `index.html` 即可。
也可以起一个本地服务器:

```bash
npx serve .
```

## 操作方式

### 电脑(键盘 / 鼠标)

| 操作 | 功能 |
| --- | --- |
| ← → / A D | 移动挡板 |
| 鼠标移动 | 移动挡板(悬停即控,无需按住) |
| 空格 / 点击画面 | 发球;激光模式下开火 |
| P | 暂停 / 继续 |
| Enter | 开始 / 重新开始 |

### 平板 / 手机(触屏)

- **手指拖动**移动挡板。
- **轻点画面**发球;激光模式下开火。
- 标题 / 结束界面:轻点即开始。

## 玩法说明

- 用挡板反弹小球击碎上方全部砖块即可过关;球落到挡板下方则损失一条命。
- 共 3 条命,全部用完游戏结束;最高分会保存在浏览器本地。
- 砖块有 1~3 点耐久,颜色区分(蓝 → 金 → 红),多耐久砖需多次击打,掉血会变色。
- 挡板反弹角度由击中位置决定:正中直上直下,越靠边缘角度越大——用它来控制球路。
- 3 个关卡布局循环轮换,每循环一圈球速加快。

### 道具(击碎砖块随机掉落,用挡板接住)

| 道具 | 效果 |
| --- | --- |
| 绿球 · 多球 | 每个现存球分裂出 1 个新球,同屏最多 6 球 |
| 红球 · 激光 | 挡板长出双炮管,10 秒内可连续开火直接打砖 |
| 蓝球 · 变长板 | 挡板宽度 ×1.5,持续 12 秒 |

## 技术要点

- 纯 Canvas 矢量手绘图形与 WebAudio 合成音效,无任何外部素材与依赖。
- 球-砖碰撞按穿透较小的一侧反弹;挡板反弹角随击中偏移线性变化(最大 60°)。
- 关卡用 10 列字符画定义(`levels.js`),新增关卡只需加一段布局。
- `?demo=秒数` 可直接开局并快进(挡板自动跟球),用于预览截图与演示。

## 文件结构

```
index.html        入口页面
style.css         页面样式
js/audio.js       WebAudio 音效
js/levels.js      常量与关卡砖块布局
js/sprites.js     矢量绘制函数(砖块/挡板/球/激光/道具)
js/entities.js    球 / 挡板 / 砖块 / 激光 / 道具 / 粒子
js/game.js        游戏状态机、碰撞、计分、HUD、主逻辑
js/main.js        键盘 + 鼠标/触屏输入与主循环
test/simulate.js  Node 无头冒烟测试: node test/simulate.js
```
