# 公路战车 Car Battle

俯视纵向卷轴公路竞速战斗游戏(红白机 Spy Hunter 风格)。
纯 HTML5 Canvas + 原生 JavaScript 实现,零依赖、零构建,打开即玩。

## 在线试玩

https://games.leonova.xyz/car-battle/

## 本地运行

无需安装任何东西,直接用浏览器打开 `index.html` 即可。
也可以起一个本地服务器:

```bash
npx serve .
```

## 操作方式

### 电脑(键盘)

| 按键 | 功能 |
| --- | --- |
| ← → / A D | 转向(低速时转向较弱,模拟真实手感) |
| ↑ / W | 加速(松开自动巡航 160 km/h) |
| ↓ / S | 刹车 |
| 空格 / J | 机炮 |
| X / K | 发射导弹(追踪) |
| P | 暂停 / 继续 |
| Enter | 开始 / 重新开始 |

### 平板 / 手机(触屏)

- 左侧**圆盘摇杆**:左/右 = 转向,上 = 加速,下 = 刹车
- 右侧红色**开火**按钮:按住机炮连射
- 右侧 **🚀** 按钮:发射导弹,**II** 按钮:暂停
- 标题 / 结束界面:轻点游戏画面即开始

## 玩法说明

- 在无尽公路上高速穿行:**里程和击毁敌车都会得分**,最高分保存在浏览器本地。
- 耐久 100 点:撞车按速度差扣 10~35、中弹 -10、刮擦路肩持续掉耐久并强减速。
- 耐久归零损失一条命(共 3 条),重生带 2 秒无敌;全灭游戏结束。

### 车辆

| 类型 | 行为 | HP | 分数 |
| --- | --- | --- | --- |
| 民用轿车(蓝/黄) | 慢速直行的障碍,撞上去两败俱伤 | 1 | 0 |
| 黑色追击车 | 从后方加速追上,并线别你 | 3 | 200 |
| 红色炮车 | 在前方控距,向后射击 | 4 | 300 |
| 装甲卡车(精英) | 高耐久慢速,打爆必掉道具 | 12 | 800 |

### 道具(打爆装甲卡车掉落,开车压过拾取)

| 道具 | 效果 |
| --- | --- |
| 🚀 导弹 | +3 枚(上限 6),追踪最近敌车,6 倍伤害 |
| ➕ 修理 | 耐久 +40(不超过上限) |
| 🛡 护盾 | 8 秒无敌,撞车中弹都不掉耐久 |

## 技术要点

- 纯 Canvas 矢量手绘图形与 WebAudio 合成音效(含随车速变调的引擎声),无任何外部素材与依赖。
- 相对速度模型:世界滚动速度 = 玩家车速,敌车按(自身速度 − 玩家速度)在屏幕上相对移动,慢车从上方进入、快车从后方追上。
- 触屏复用圆盘虚拟摇杆(同坦克大战),并为底部控制区预留空间,iPad 横竖屏完整显示。
- 难度随里程推进:刷车间隔缩短、战斗车占比提高。
- `?demo=秒数` 自动驾驶快进、`?touch=1` 强制显示触屏控制,用于预览截图与调试。

## 文件结构

```
index.html        入口页面
style.css         页面样式 + 触屏控制区
js/audio.js       WebAudio 音效与引擎声
js/levels.js      常量(公路几何/车速)与难度曲线
js/sprites.js     矢量绘制函数(跑车/敌车/导弹/道具)
js/entities.js    玩家车 / 敌车 / 子弹 / 道具 / 粒子
js/game.js        游戏状态机、相对速度、碰撞、HUD、主逻辑
js/main.js        键盘 + 圆盘摇杆输入与主循环
test/simulate.js  Node 无头冒烟测试: node test/simulate.js
```
