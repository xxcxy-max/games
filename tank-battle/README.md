# 坦克大战 Tank Battle

经典 FC / 小霸王风格《坦克大战》(Battle City) 的网页复刻版。
纯 HTML5 Canvas + 原生 JavaScript 实现，零依赖、零构建，打开即玩。

## 在线试玩

https://games.leonova.xyz/tank-battle/

## 本地运行

无需安装任何东西，直接用浏览器打开 `index.html` 即可。
也可以起一个本地服务器：

```bash
npx serve .
```

## 操作方式

### 电脑(键盘)

| 按键 | 功能 |
| --- | --- |
| 方向键 / WASD | 移动坦克 |
| 空格 / J | 射击 |
| P | 暂停 / 继续 |
| Enter | 开始 / 重新开始 |

### 平板 / 手机(触屏)

触屏设备上画面下方会自动出现虚拟按键：

- 左侧**十字键**：按住移动
- 右侧红色**开火**按钮：按住连续射击
- 中间小圆钮 **II**：暂停 / 继续
- 标题 / 结束界面：轻点游戏画面即开始

## 玩法说明

- 驾驶黄色坦克，消灭每一关的全部敌方坦克即可过关。
- 保护好屏幕底部中央的基地(老鹰)：基地被任何子弹击中，游戏立即结束。
- 你有 3 条命，全部阵亡游戏结束；阵亡后会在出生点带保护罩重生。
- 过关后敌人数量与速度提升，4 个关卡循环轮换。

### 地形

- **砖墙**：可被子弹打掉，打出的缺口会改变战场格局(包括基地的保护墙)。
- **钢墙**：无法摧毁。
- **河水**：坦克无法通过，但子弹可以飞过。
- **树林**：可以通行，但会遮挡其中的坦克，适合伏击。

### 敌方坦克

| 类型 | 特点 | 分数 |
| --- | --- | --- |
| 灰色普通型 | 速度慢、子弹慢 | 100 |
| 绿色快速型 | 移动快 | 200 |
| 金色火力型 | 子弹快 | 300 |

## 技术要点

- 纯 Canvas 手绘像素风图形与 WebAudio 合成音效，无任何外部素材与依赖。
- 26×26  tile 地图，砖墙按小方块粒度被子弹逐步破坏。
- 敌方坦克 AI：随机游走 + 朝下进攻倾向 + 随机开火。
- 经典的转向半格对齐手感、子弹对撞抵消、出生无敌保护罩。

## 文件结构

```
index.html        入口页面
style.css         页面样式
js/audio.js       WebAudio 音效
js/levels.js      常量与关卡地图数据
js/sprites.js     像素风绘制函数
js/entities.js    坦克 / 子弹 / 特效等实体
js/game.js        游戏状态机、碰撞、主逻辑
js/main.js        键盘输入与主循环
test/simulate.js  Node 无头冒烟测试: node test/simulate.js
```
